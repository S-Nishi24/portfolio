# portfolio

## アプリ概要
Next.js と Supabase を用いた「旅行支出管理・天気確認アプリ」です。
ユーザーは旅行予定を登録し、その地点の天気情報を確認できます。
登録した支出情報の内訳や使用状況をグラフで確認できます。
